7.4.9. getItems

7.4.9.1. Synopsis

AITResult getItems(integer $offset=null, integer $lines=null, integer $ordering=null)

7.4.9.2. Description

Retourne tous les items du type d'item courant

7.4.9.3. Paramètres

integer $offset

décalage à partir du premier enregistrement

integer $lines

nombre de lignes à retourner

integer $ordering

flag permettant le tri

7.4.9.4. Retour

un objet AITResult (ArrayObject) contenant des objets AIT_Item.

7.4.9.5. Exemple

<?php
require_once 'prepend.php';

$items = $schema->disques->getItems();


foreach ($items as $item) {
    echo "- ".$item->get()."\n";
}
echo 'Showed : '.$items->count(). ' / '.$items->total()."\n";

// Affichera :
//
// - a2407474
// - a2274830
// - a1902425
// - a762424
// - a1095444
// (...)
