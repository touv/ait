8.6.8. searchTags

8.6.8.1. Synopsis

AITResult searchTags(mixed $query, integer $offset=null, integer $lines=null, integer $ordering=null array $cols=[])

8.6.8.2. Description

Recherche des tags du type courant

8.6.8.3. Paramètres

string $query

requete (le format peut varier en focntion de la callback) sans callback c'est une chaine de caractères contenant du SQL

integer $offset

décalage à partir du premier enregistrement

integer $lines

nombre de lignes à retourner

integer $ordering

flag permettant le tri

array $cols

filtre sur les champs complémentaires

8.6.8.4. Retour

un objet AITResult (ArrayObject) contenant des objets AIT_Tag.
