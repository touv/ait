defTag


Synopsis
++++++++

    AIT_Tag defTag(string $l, AIT_TagType $o)

Description
+++++++++++

Retourne un tag associé à l'item courant.
Si le Tag n'existe pas il est automatiquement créé.

Paramètres
++++++++++

string *$l*:: nom du tag
AIT_TagType *$o*:: Type de Tag


Retour
++++++

Un objet AIT_Tag représentant le tag.
