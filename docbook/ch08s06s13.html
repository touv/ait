8.6.13. getSystemID

8.6.13.1. Synopsis

integer  getSystemID()

8.6.13.2. Description

Retourne l'identifiant système de l'élément.

8.6.13.3. Paramètres

Aucun.

8.6.13.4. Retour

Un nombre entier représentant l'identifiant système.

<?php
require_once 'prepend.php';

$item = $schema->disques->getItem('a1902425');
$id = $item->getSystemID();

var_dump($id);

// Affichera : int(17)
