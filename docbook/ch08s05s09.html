8.5.9. getItems

8.5.9.1. Synopsis

AITResult getItems(integer $offset=null, integer $lines=null, integer $ordering=null, array $cols=[])

8.5.9.2. Description

Retourne tous les items attachés à l'item courant

8.5.9.3. Paramètres

integer $offset

décalage à partir du premier enregistrement

integer $lines

nombre de lignes à retourner

integer $ordering

flag permettant le tri

array $cols

filtre sur les champs complémentaires

8.5.9.4. Retour

Un objet AITResult (ArrayObject) contenant des objets AIT_Item.
