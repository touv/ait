7.6.6. countItems

7.6.6.1. Synopsis

AIT_TagType::countItems()

7.6.6.2. Description

Retourne le nombre total de tags attachés au type de tag courrant.

7.6.6.3. Paramètres

Aucun.

7.6.6.4. Retour

un nombre entier représentant le nombre d'enregistrement.

<?php
require_once 'prepend.php';

echo $schema->style->countItems();
