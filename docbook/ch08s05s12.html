8.5.12. fetchItems

8.5.12.1. Synopsis

AITResult fetchElements(ArrayObject $types, integer $offset=null, integer $lines=null, integer $ordering=null, array $cols=[], $control = null)

8.5.12.2. Description

Retourne les élements attachés à l'item courant à condition qu'il soit de l'un des types d'item donné dans $types

8.5.12.3. Paramètres

ArrayObject $tags

Tableau de type d'item

integer $offset

décalage à partir du premier enregistrement

integer $lines

nombre de lignes à retourner

integer $ordering

flag permettant le tri

array $cols

filtre sur les champs complémentaires

integer $control

permet de filtrer suivant les tags (AIT::TAG) ou les items (AIT::ITEM)

8.5.12.4. Retour

Un objet AITResult (ArrayObject) contenant des objets AIT_Item et/ou AIT_Tag.
