8.4.2. defTagType

8.4.2.1. Synopsis

AIT_TagType defTagType(string $l)

8.4.2.2. Description

Retourne un type de tag associé au type d'item courant. Si le Tag n'existe pas il est automatiquement créé.

8.4.2.3. Paramètres

string $l

label

8.4.2.4. Retour

Un objet AIT_TagType représentant le type de tag
