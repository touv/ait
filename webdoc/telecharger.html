Télécharger et Installer AIT

Package PEAR

Channel PEAR

Le projet Pxxo met à disposition dans son channel AIT.

$ pear channel-discover pear.pxxo.net
$ pear install pxxo/ait

Fichier TGZ

On peut télécharger le package PEAR sous forme de fichier TGZ à cette adresse : http://pear.pxxo.net/get/

Code Source

Le code source de AIT est disponible sur Github.com : http://github.com/touv/ait/tree/master
