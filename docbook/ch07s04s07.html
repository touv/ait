7.4.7. getItem

7.4.7.1. Synopsis

AIT_Item getItem(string $l)

7.4.7.2. Description

Récupère un item

7.4.7.3. Paramètres

string $l

nom de l'item

7.4.7.4. Retour

Un objet AIT_Item représentant l'item trouvé.

7.4.7.5. Exemple

<?php
require_once 'prepend.php';

$item = $schema->disques->getItem('a1902425');
if (!is_null($item)) {
    echo $item->get();
}
// Affichera :  a1902425
