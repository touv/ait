8.4.14. getItemBySystemID

8.4.14.1. Synopsis

AIT_Item getItemBySystemID(integer $i)

8.4.14.2. Description

Récupère un Item en fonction de son identifiant système

8.4.14.3. Paramètres

integer $i

un identifiant système

8.4.14.4. Retour

Un objet AIT_Item représentant l'item trouvé.

8.4.14.5. Exemple

<?php
require_once 'prepend.php';

$itemA = $schema->disques->getItem('a1902425');
$id = $itemA->getSystemID();

$itemB = $schema->disques->getItemBySystemID($id);

if (!is_null($itemB)) {
    echo $itemB->get();
}
// Affichera :  a1902425
