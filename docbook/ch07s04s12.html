7.4.12. searchItems

7.4.12.1. Synopsis

AITResult searchItems(mixed $query, integer $offset=null, integer $lines=null, integer $ordering=null)

7.4.12.2. Description

Recherche des items du type courant

7.4.12.3. Paramètres

string $query

requete (le format dépend de la search_callback) sans callback c'est une chaine de caractères contenant du SQL

integer $offset

décalage à parir du premier enregistrement

integer $lines

nombre de lignes à retourner

integer $ordering

flag permettant le tri

7.4.12.4. Retour

un objet AITResult (ArrayObject) contenant des objets AIT_Item.
