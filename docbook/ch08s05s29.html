8.5.29. getBySystemID

8.5.29.1. Synopsis

mixed AIT::getBySystemID(PDOAIT $pdo, $id)

8.5.29.2. Description

Créé l'objet correspondant à un identifiant système

8.5.29.3. Paramètres

PDOAIT $pdo

pointeur sur la base de données

integer $id

identifiant systéme

8.5.29.4. Retour

Retourne NULL ou une instance d'objet (AIT_ItemType, AIT_Item, AIT_TagType, AIT_Tag)

8.5.29.5. Exemple

<?php
require_once 'prepend.php';

$o = AIT::getBySystemID($db, 17);
echo $o->get() ." : ";

$o = AIT::getBySystemID($db, 18);
echo $o->get() .", ";

$o = AIT::getBySystemID($db, 19);
echo $o->get() ."\n";
