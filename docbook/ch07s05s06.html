7.5.6. getTags

7.5.6.1. Synopsis

AITResult getTags(integer $offset=null, integer $lines=null, integer $ordering=null)

7.5.6.2. Description

Retourne tous les tags de l'item courant

7.5.6.3. Paramètres

integer $offset

décalage à partir du premier enregistrement

integer $lines

nombre de lignes à retourner

integer $ordering

flag permettant le tri

7.5.6.4. Retour

Un objet AITResult (ArrayObject) contenant des objets AIT_Tag.
