8.5.15. getTagsObject

8.5.15.1. Synopsis

AITTagsObject getTagsObject()

8.5.15.2. Description

Retourne tous les tags de l'élement courant sous forme d'un objet.

8.5.15.3. Paramètres

Aucun.

8.5.15.4. Retour

Un objet AITTagsObject représentant un ensemble de tag trié par type

8.5.15.5. Exemple

<?php
require_once 'prepend.php';

$item = $schema->disques->getItem('a1902425');

$tct = $item->getTagsObject();

echo $tct->titre->offsetGet(0)->get() . "\n";
# Affichera : Heroes

foreach($tct->artiste as $tag)
    echo ' * ' .$tag->get();

# Affichera : * Antonio Vivaldi * Philippe Jaroussky
