7.4.21. getSystemID

7.4.21.1. Synopsis

integer  getSystemID()

7.4.21.2. Description

Retourne l'identifiant système de l'élément.

7.4.21.3. Paramètres

Aucun.

7.4.21.4. Retour

Un nombre entier représentant l'identifiant système.

<?php
require_once 'prepend.php';

$item = $schema->disques->getItem('a1902425');
$id = $item->getSystemID();

var_dump($id);

// Affichera : int(17)
