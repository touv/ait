8.5.13. del

8.5.13.1. Synopsis

void del(boolean $cascade = false)

8.5.13.2. Description

Suppression de l'élément. On peut également avec le paramètre $cascade supprimer les tags associés.

8.5.13.3. Paramètres

boolean $cascade

permet une suppression en cascade

8.5.13.4. Retour

NULL.
