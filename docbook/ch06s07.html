6.7. AIT_Tag

Cette classe permet de manipuler un TAG.

6.7.1. __construct

6.7.1.1. Synopsis

void __construct($l, $t, $i, PDOAIT $pdo, $id=false)

6.7.1.2. Description

Le constructeur de la classe.

6.7.1.3. Paramètres

string $l

Label du TAG

integer $t

Identifiant système du TYPE de TAG associé.

integer $i

Identifiant système de l'ITEM (si connu).

PDOAIT $pdo

Instance de base AIT que l'on souhaite utiliser.

integer $id

Identifiant système de l'élément (si déjà connu).

6.7.2. detach

6.7.2.1. Synopsis

AIT_Tag detach()

6.7.2.2. Description

Supprime l'association entre le tag son item

6.7.2.3. Paramètres

Aucun.

6.7.2.4. Retour

Retourne l'objet en lui-même. Ce qui permet d'enchainer les méthodes.

6.7.2.5. Exemple

<?php
require_once 'prepend.php';

$item = $schema->disques->getItem('a1902425');

$tags = $item->getTypedTags($schema->style);

foreach($tags as $tag) {
    $tag->detach();
}
// Supprime tous les tags "style" asscoié à a1902425

6.7.3. attach

6.7.3.1. Synopsis

AIT_Tag attach(AIT_Item $o)

6.7.3.2. Description

Ajoute une association entre le tag et un item

6.7.3.3. Paramètres

AIT_Item $o

Un objet contenant un item.

6.7.3.4. Retour

Retourne l'objet en lui-même. Ce qui permet d'enchainer les méthodes.

<?php
require_once 'prepend.php';

$item = $schema->disques->getItem('a1902425');

$tag = $schema->style->addTag('disco');

// Ajout du tag "disco" à "a1902425"
$tag->attach($item);

6.7.13. getTimestamps

6.7.13.1. Synopsis

ArrayObject  getTimestamps()

6.7.13.2. Description

Retourne la date de dernière modification et la date de création de l'élément. les différents "temps" enregistré pour l'élément

6.7.13.3. Paramètres

Aucun.

6.7.13.4. Retour

Un objet ArrayObject dont le premier item contient la date de modification et le deuxième contient la date de création.

6.7.13.5. Exemple

<?php
require_once 'prepend.php';

$item = $schema->disques->getItem('a1902425');
$timestamps = $item->getTimestamps();
echo date('Y-m-d\TH:i:s\Z', $timestamps->updated)."\n";
// Affichera : 2008-08-21T17:22:17Z
echo $timestamps['created']."\n";
// Affichera : 1219332137


$jazz = $schema->style->getTag('Jazz');
$timestamps = $jazz->getTimestamps();
var_dump($timestamps);

// Affichera :
//  object(ArrayObject)#15 (2) {
//     ["updated"]=>
//     int(1219332137)
//     ["created"]=>
//     int(1219332137)
//  }
//

6.7.17. dump

6.7.17.1. Synopsis

mixed dump(string $s = '', boolean $r = false)

6.7.17.2. Description

Afficher le contenu de l'objet. Si $r est TRUE. Alors le contenu de l'objet précédé éventuellement de $s seront affichée entouré des <pre> et </pre>

6.7.17.3. Paramètres

string $s

chaine de carat cure à afficher avant le contenu de l'objet.

booelan $r

si vrai a retourne la chaine à afficher plutôt que de l'afficher

6.7.17.4. Retour

Retourne NULL ou une chaine de caractères (dépend du paramètre $r).
