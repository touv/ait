7.7.12. getSystemID

7.7.12.1. Synopsis

integer  getSystemID()

7.7.12.2. Description

Retourne l'identifiant système de l'élément.

7.7.12.3. Paramètres

Aucun.

7.7.12.4. Retour

Un nombre entier représentant l'identifiant système.

<?php
require_once 'prepend.php';

$item = $schema->disques->getItem('a1902425');
$id = $item->getSystemID();

var_dump($id);

// Affichera : int(17)
