6.3. AITQuery

Cette classe permet de créer des requêtes avancées pour trouver des items.

6.3.1. __construct

6.3.1.1. Synopsis

function __construct(AITPDO $pdo)

6.3.1.2. Description

Le constructeur de la classe.

6.3.1.3. Paramètres

AITPDO $pdo

L'instance de base AIT que l'on souhaite interroger.

6.3.2. all

6.3.2.1. Synopsis

public function all(ArrayObject $tags)

6.3.2.2. Description

On recherche des items ayant tous les tags donnés en paramètre.

6.3.2.3. Paramètres

$tags

Contient des objet AIT_Tag représentant les tags que l'on souhaite rechercher

6.3.2.4. Retour

Retourne FALSE si aucun tag n'a été trouvé dans le tableau d'entrée sinon retourne TRUE.

6.3.3. one

6.3.3.1. Synopsis

public function one(ArrayObject $tags)

6.3.3.2. Description

On recherche des items ayant au moisn l'un des tags donnés en paramètre.

6.3.3.3. Paramètres

$tags

Contient des objet AIT_Tag représentant les tags que l'on souhaite rechercher

6.3.3.4. Retour

Retourne false si aucun tag n'a été trouvé dans le tableau d'entrée sinon retourne true.

6.3.4. getSQL

6.3.4.1. Synopsis

public function getSQL()

6.3.4.2. Description

Retourne la requête construite avec les méthodes one et all sous forme de requête SQL.

6.3.4.3. Paramètres

Aucun.

6.3.4.4. Retour

Retourne une chaine de caractère contenant une requête SQL.
