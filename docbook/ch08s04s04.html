8.4.4. getTagType

8.4.4.1. Synopsis

AIT_TagType getTagType(string $l)

8.4.4.2. Description

Retourne un type de tag du type d'item courant en fonction de son label.

8.4.4.3. Paramètres

string $l

Le label.

8.4.4.4. Retour

Un objet AIT_TagType représentant le type de tag trouvé ou NULL si il n'existe pas.
