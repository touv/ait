8.4.16. countItems

8.4.16.1. Synopsis

AIT_ItemType::countItems()

8.4.16.2. Description

Retourne le nombre total d'items attachés au type d'item courrant.

8.4.16.3. Paramètres

Aucun.

8.4.16.4. Retour

un nombre entier représentant le nombre d'enregistrement.

8.4.16.5. Exemple

<?php
require_once 'prepend.php';

$counter = $schema->disques->countItems();

echo $counter;

// Affiche le nombre de disque
