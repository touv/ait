8.6.14. get

8.6.14.1. Synopsis

mixed  get(string $name= 'label')

8.6.14.2. Description

Retourne la valeur d'une propriété de l'élément. Les propriétés accessibles sont : type, label, buffer, prefix, suffix

8.6.14.3. Paramètres

string $name

nom de la propriété

8.6.14.4. Retour

mixed
